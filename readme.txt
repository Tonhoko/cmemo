■ このソフトウェアについて

    このソフトウェアは、デスクトップにメモを貼り付けるアプリケーションです。
    
        [特徴]
        - テキストのみのシンプルな機能
        - 外部テキストエディタ
        - キーボードだけで操作が出来る
        - GoogleDocsを使って複数のPC間で同期できる
        - Pythonを使ったカスタマイズが出来る


■ 配布 / 保障について

    - このソフトウェアを二次配布する場合は、作者に事前連絡してください。
    
    - このソフトウェアを改造して配布してもかまいませんが、readme文書や
      ドキュメントの中に、オリジナル版の配布元への参照を記述してください。

    また、本ソフトウェアの使用によるいかなる損害も保障しかねますので、
    プログラムの実行は本人の責任で行ってください。


■ ファイルとインストールについて

    配布されたファイルを解凍すると以下のファイルとフォルダが作成されます。

    cmemo/
        cmemo.exe --------- プログラム本体
        python33.dll ------ Pythonランタイムライブラリ
        _config.py -------- デフォルト設定ファイル
        readme.txt -------- この文書
        data/ ------------- メモデータ格納ディレクトリ
        backup/ ----------- バックアップデータ格納ディレクトリ
        doc/ -------------- 説明書
        extension/ -------- ユーザサイドのスクリプトを入れるディレクトリ
        license/ ---------- このソフトウェアで使用しているサードパーティ
                            ライブラリのライセンス文書
        src.zip ----------- ソースコード

    また、最初の起動で自動的に以下のファイルが作られます。

        config.py --------- 設定ファイル
        cmemo.ini --------- 状態保存ファイル

    _config.py は デフォルトの設定ファイルです。
    上書きインストールによって、ユーザの設定ファイルが不用意に上書きされ
    ないように先頭に _ が付加されています。
    最初の起動で、_config.py が config.py に自動的にコピーされます。


■ アンインストール

    インストールしたフォルダごと消してください。
    このソフトウェアはレジストリを使用しません。
